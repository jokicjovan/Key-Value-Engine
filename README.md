# Key-Value-Engine
High performance oriented implementation of algorithms and data structures in Golang. Engine parameters could be configured from userConfiguration.json file located at the data directory.

## Data structures
- LRU Cache
- LSM Tree
- SSTable
- Token Bucket
- Write Ahead Log
- Bloom Filter
- Count Min Sketch
- Hyper Log Log
- Memtable
- Merkle Tree
- SkipList
